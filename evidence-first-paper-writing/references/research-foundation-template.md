# 研究基础模板

在形成提纲或起草正文前使用。

| 字段 | 需要填写的内容 | 状态 |
|---|---|---|
| 研究问题 | 一个可被研究回答的问题 | |
| 研究目标或假设 | 可检验的目标，并注明是否为探索性研究 | |
| 实践与学术缺口 | 现有研究未充分解决之处，并给出来源 | |
| 拟主张的贡献 | 仅限本研究能够支撑的贡献 | |
| 研究对象 | 人群、数据集、系统、语料或材料 | |
| 已有证据 | 数据、研究方案、图、日志、已核验文献 | |
| 缺失证据 | 提出核心主张前仍需补充的材料 | |
| 方法与比较方案 | 流程、基线/对照、指标、统计方法 | |
| 约束 | 样本、时间、伦理、算力、访问权限、范围 | |
| 目标期刊/会议要求 | 文种、篇幅、模板、披露要求 | |

## 主张账本

| 主张 | 主张类型 | 支撑证据 | 局限 | 目标章节 | 状态 |
|---|---|---|---|---|---|
| | 文献 / 方法 / 结果 / 解释 | | | | 已核验 / 待补 / 不支持 |

作者必须确认或补充的内容使用 `AUTHOR_INPUT_NEEDED` 标记。对于无证据支持的主张，应删除或缩小表述范围。
