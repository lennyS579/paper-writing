2. 对外发布本工作流时，保留此表和各上游项目的许可证说明；如实际复制了任何上游文本、脚本或资产，再按其原许可证补充相应 NOTICE。
3. GitHub 链接反映本机材料与公开仓库的核验结果。上游项目、作者归属和许可证可能更新，应在发布前再次核验。
